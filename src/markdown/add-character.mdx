## 인물 생성하기

### 이번 가이드에서는 인물을 생성하는 과정에 대해서 알아봅니다.

![character-sidebar](/guide-image/character-sidebar.png)

인물을 생성할 프로젝트를 선택한 후, 왼쪽의 사이드바에서 인물 - 인물 관리를 클릭하여, 인물 리스트 페이지로 전환합니다.

![character-list](/guide-image/character-list.png)

그 후 새로운 인물 생성을 눌러 인물 폼 페이지로 전환합니다.

![character-form](/guide-image/character-form.png)

각 항목을 모두 입력한 후 생성을 누릅니다. 새로운 인물를 생성할 시에는 토큰이 소모되지 않습니다.
