## 프로젝트 생성하기

> 이번 가이드에서는 새로운 프로젝트를 생성하는 과정에 대해서 알아봅니다.

![project-sidebar](/guide-image/project-sidebar.png)

왼쪽의 사이드바에서 프로젝트 목록을 클릭하여, 프로젝트 리스트 페이지로 전환합니다.

![project-navigation](/guide-image/project-navigation.png)

그 후 새로운 프로젝트 생성을 눌러 프로젝트 폼 페이지로 전환합니다.

![project-form](/guide-image/project-form.png)

각 항목을 모두 입력한 후 생성을 누릅니다. 이때, 새로운 프로젝트를 생성할 시 50토큰이 소모됩니다.
