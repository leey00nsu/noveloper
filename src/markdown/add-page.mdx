## 페이지 생성하기

> 이번 가이드에서는 페이지를 생성하는 과정에 대해서 알아봅니다.

![page-sidebar](/guide-image/page-sidebar.png)

페이지를 생성할 프로젝트를 선택한 후, 왼쪽의 사이드바에서 스토리 - 페이지 관리를 클릭하여, 페이지 리스트 페이지로 전환합니다.

![page-list](/guide-image/page-list.png)

그 후 새로운 페이지 생성을 눌러 페이지 폼 페이지로 전환합니다.

![page-form](/guide-image/page-form.png)

![page-ai-option](/guide-image/page-ai-option.png)

AI 생성 버튼을 통해 작성된 페이지 내용을 기반으로 AI에게 도움을 받을 수 있습니다.

현재 커서 위치에 한 문장을 생성하거나 한 문단(3 문장)을 생성할 수 있습니다.

![page-ai-example-1](/guide-image/page-ai-example-1.png)

또한 현재 선택된 내용을 반으로 줄이는 요약을 할 수 있습니다.

![page-ai-example-3](/guide-image/page-ai-example-3.png)

페이지 요약에서 AI 생성 버튼을 클릭하면, 현재 페이지 내용을 기반으로 페이지 요약이 생성됩니다.

![page-ai-example-2](/guide-image/page-ai-example-2.png)

각 항목을 모두 입력한 후 생성을 누릅니다. 이때, 새로운 페이지를 생성할 시에는 20토큰이 소모됩니다.

## 페이지 내보내기

텍스트로 내보내기를 클릭하여 페이지 내용을 txt 파일로 내보낼 수 있습니다.

![page-export-txt](/guide-image/page-export-txt.png)

pdf로 내보내기를 클릭하여 페이지 내용을 pdf 파일로 내보낼 수 있습니다.

![page-export-pdf](/guide-image/page-export-pdf.png)
