## 인물 관계도 설정하기

> 이번 가이드에서는 인물 관계도를 설정하는 과정에 대해서 알아봅니다.

![relation-sidebar](/guide-image/relation-sidebar.png)

인물 관계도는 좌측 사이드바에서 인물 관계도를 클릭하여 전환할 수 있습니다.

![relation-1](/guide-image/relation-1.png)

다음과 같이 인물을 생성하게 되면, 현재 존재하는 인물 목록에 대해 자동으로 관계도에 인물이 추가됩니다.

![relation-2](/guide-image/relation-2.png)

생성된 인물을 드래그하여 위치를 변경할 수 있으며, 한 번 클릭하여 관계를 연결할 수 있는 상태로 변경할 수 있습니다.

![relation-3](/guide-image/relation-3.png)

해당 상태에서 점을 다른 인물에 드래그하게 되면 관계가 생성됩니다.

|                                            |                                            |                                            |
| ------------------------------------------ | ------------------------------------------ | ------------------------------------------ |
| ![relation-4](/guide-image/relation-4.png) | ![relation-5](/guide-image/relation-5.png) | ![relation-6](/guide-image/relation-6.png) |

관계를 클릭하여 원하는 관계를 입력할 수 있습니다.

만약, 인물이 삭제되면 관련된 관계가 모두 사라지니 인물을 삭제할 때는 주의해야 합니다.
